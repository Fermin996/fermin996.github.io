About Me

I'm a Full Stack Web Developer specializing in the MERN stack. I have been creating web apps since 2019 when I completed Ironhack's web development bootcamp.

I spend most of my free time developing personal projects that interest me and help me improve as a software engineer. The project Waffl was created as a result of years of working with POS (point of sale) applications, which lacked features that I wished to implement in my own POS application.

Currently i'm learning Swift and React Native with the hopes of breaking into mobile development. My long term goal is to improve my software engineering skills to provide value as an engineer and become a team leader.
